# ManufacturingDowntime_DEPI_Project

## 📊 Project Overview
This project analyzes manufacturing downtime data to identify key causes of production interruptions, measure their impact, and provide actionable insights to improve operational efficiency.

The goal is to reduce unplanned downtime, optimize maintenance planning, and improve overall equipment effectiveness (OEE).

---

## 🎯 Objectives

- Measure total downtime across machines and production lines
- Identify top causes of downtime
- Analyze downtime trends over time
- Compare downtime across shifts
- Evaluate maintenance effectiveness
- Provide data-driven recommendations

---

## 🏭 Business Problem

Unplanned downtime leads to:
- Production delays
- Increased operational costs
- Lower productivity
- Missed delivery deadlines

This analysis aims to detect patterns and root causes to support management decisions.

---

## 📂 Dataset Description

The dataset includes:

- Machine ID
- Production Line
- Downtime Duration (minutes)
- Downtime Cause
- Shift
- Date
- Maintenance Type
- Operator

---

## 🧹 Data Cleaning Process

- Handling missing values
- Standardizing downtime causes
- Converting date columns
- Removing duplicates
- Detecting and handling outliers

---

## 📈 Exploratory Data Analysis (EDA)

Key analysis performed:

- Total downtime per machine
- Downtime by production line
- Top 5 downtime causes
- Monthly downtime trend
- Downtime by shift
- Correlation analysis

---

## 📊 Key KPIs

- Total Downtime (Minutes)
- Average Downtime per Incident
- Downtime Frequency
- Mean Time Between Failures (MTBF)
- Mean Time To Repair (MTTR)
- Downtime by Shift %
- Downtime by Cause %

---

## 📌 Key Insights

- Identified machines with highest downtime
- Detected most frequent failure causes
- Observed peak downtime periods
- Found shifts with higher interruption rates

## 🚀 Recommendations

- Implement preventive maintenance for high-risk machines
- Schedule maintenance during low-production shifts
- Provide additional operator training
- Monitor top downtime causes monthly
