# ManufacturingDowntime_DEPI_Project
A comprehensive Data Analytics project that analyzes manufacturing downtime, production performance, and operational efficiency using SQL Server, Python, Power BI, and Tableau.
The project follows the complete Data Analytics lifecycle, from raw manufacturing data to interactive dashboards that support data-driven decision-making.

## рџ“Љ Project Overview
Manufacturing downtime is one of the most significant challenges affecting production efficiency, operational costs, and equipment utilization.

This project analyzes manufacturing downtime data to identify the key causes of production interruptions, measure their operational impact, and provide actionable insights to improve productivity and optimize maintenance planning.

The final solution combines SQL Server, Python, Power BI, and Tableau to transform raw manufacturing data into meaningful business intelligence.

## рџЋЇ Objectives

- Measure total downtime across machines and production lines
- Identify top causes of downtime
- Analyze downtime trends over time
- Compare downtime across shifts
- Evaluate maintenance effectiveness
- Provide data-driven recommendations
- Support business decisions using interactive dashboards
- Improve Overall Equipment Effectiveness (OEE)
